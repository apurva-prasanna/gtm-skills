# GTM Skills

A collection of AI tools for enterprise GTM in APAC — built and documented as part of [Narratives & Nodes](https://apurvap.substack.com), a build-in-public series on AI in complex B2B sales.

Every tool here is built around one thesis: the context gap is the biggest unsolved problem in enterprise GTM, and AI can close it.

---

## Tools

### MEDDPICC Qualifier
A Flask app for adtech and media sales. Enter a client name and market — it researches the account across multiple angles and produces a structured MEDDPICC qualification document with Research tags, To Validate flags, and sources per field.

Built for brand-direct and agency-led deals. Detects deal type from context.

→ [meddpicc-qualifier/](./meddpicc-qualifier)

---

### GTM Narrative Partner
An AI-powered narrative and story partner for enterprise B2B sales and client services teams. Works in two modes — proactive (market/trend-led storytelling) and reactive (client-specific narrative with MEDDIC discovery).

Available as:
- **Claude Skill** — copy the folder into your skills directory for use as a slash command in Claude Code
- **Claude Project instructions** — paste into your Claude Project for use in claude.ai
- **Microsoft Copilot Agent instructions** — reference and edit in your own Copilot Agent Builder instance

→ [gtm-narrative-partner/](./gtm-narrative-partner)
